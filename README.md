


Retail Inventory Twin


Predicts days until stock-out for retail products using Prophet time-series forecasting and weather features.


Prerequisites：

1.Google account (for Colab)
2.Python 3.x

Installation:

1.Open Google Colab at colab.research.google.com
2.Upload retail_store_inventory.csv to the Files panel
3.Run the first cell to install dependencies:
!pip install prophet pandas matplotlib scikit-learn

Usage:

1.Run all cells in order
2.View MAE/RMSE results for all 20 products
3.View days until stock-out predictions

Troubleshooting:

1.If prophet install fails, restart runtime and try again
2.If KeyError appears, make sure CSV file is uploaded correctly

Requirements:

1.prophet
2.pandas
3.matplotlib
